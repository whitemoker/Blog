环境要求

  1. Node.js

    • 版本要求：>= 12.0
    • 下载地址：https://nodejs.org
    • 安装后会自动包含 npm 包管理器

  2. Hexo CLI

技术栈详解

  1. 核心环境

    • Node.js v22.12.0 (运行环境)
    • npm v10.8.3 (包管理器)
    • Hexo CLI (命令行工具)

  2. 博客框架

    • Hexo v4.3.2 (静态博客生成器)
    • hexo-toc (文章目录生成插件)

  3. 部署平台

    • GitHub Pages (静态网站托管)
    • Git (版本控制)

  4. 内容格式

    • Markdown (文章编写格式)
    • Front Matter (文章元数据配置)
