常见问题

1. npm安装失败

  • 检查Node.js是否正确安装
  • 尝试使用管理员权限运行命令
  • 检查网络连接

2. hexo命令未找到

  • 确保全局安装了hexo-cli
  • 重新运行 npm install -g hexo-cli

3. 文章不显示

  • 确保文章front-matter格式正确
  • 运行 hexo clean && hexo generate
  • 检查文件编码是否为UTF-8
